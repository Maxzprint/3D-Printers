copy the prusaslicer folder and paste it in c drive.

Open the the folder in c drive and right click "prusa-slicer" and choose send to - desktop. 
This will create a shortcut to start the program.

Run the slicer, if the setup wizard starts, cancell it.

Click File - Import _ Config Bundle, choose the config bundle downloaded with this program.

You're good to go, if this is your first time using prusa slicer, 
check out the videos by Chris's Basement, Tom's Guide and Joseph Prusa
